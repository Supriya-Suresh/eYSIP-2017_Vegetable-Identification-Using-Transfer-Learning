# Vegetable Identification Using Transfer Learning
This Project aims to help automate the logging of farm produce. Vegetables are identified using Neural Networks and automatically monitored which helps reduce manual labor.
You can find more information about this project on our [wiki page](https://github.com/eYSIP-2017/eYSIP-2017_Vegetable-Identification-Using-Transfer-Learning/wiki).
